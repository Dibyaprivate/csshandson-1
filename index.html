what is css?

  1. CSS stands for cascading style sheets.

  2. CSS defines how html elements are to be display on screen,paper,or in other media

  3. Css is not necesorycss saves a lot of work,it can cantrol the layout of multiple web pages all at once

CSS can be added to HTML documents in 3 ways:

  • inline- by using the style attribute inside HTML elements
  • internal-by using a style element in the head section
  • External-by using a link element to link to an external CSS file.
  • The most common way to add CSS. is to keep the styles in external CSS files.However,in this tutotrial we will use inline and internal styles,because this is easier to demonstrate,and easier for youto try it yourself.

when comparing CSS class vs id.the difference is that css class applies a style to multiple elements.ID.on the other hand,applies a style to one unique element.ID is also special in that you can use a special URL directly to an element and it's used by javascript.

In CSS,selectors are used to target a specific element or range of elements on a web page.Once an element has been target,a style or set of styles can be applied to the element.

There is a wide range of selectors available.Two of the most commonly used are class and ID.Both are used to target elements to which a style should be applied.
